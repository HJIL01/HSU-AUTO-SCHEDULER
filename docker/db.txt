해당 학기의 전공들을 담을 테이블도 만들어야함

학기 테이블 (
    학기 id(pk, 학기년도로 이루어진 문자열), 
    년도, 
    학기
)

전공 테이블 (
    전공 id, 
    전공 이름
)

학기_전공 관계 테이블 (
    학기 id,
    전공 id
)

과목 테이블 (
    과목 id(pk, 과목코드분반으로 이루어진 문자열), 
    과목 코드, 
    과목명, 
    이수 구분, 
    과목 구분, 
    학점, 
    주야, 
    분반, 
    학년, 
    학년 제한, 
    온라인 시간: number
)

학기_과목 관계 테이블 (
    학기 id,
    과목 id
)

교수 테이블 (
    교수 id(pk),
    이름,
    학과,
    이메일
)

오프라인 시간표 테이블 (
    오프라인 시간표 id(pk, 과목id요일시작시간 합친 문자열)
    과목 id(fk),
    요일,
    시작 시간,
    끝나는 시간, 
    장소
)

중심 관계 테이블 (
    학기 id,
    전공 id,
    과목 id,
    교수 id
)
