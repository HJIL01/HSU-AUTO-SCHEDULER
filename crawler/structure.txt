챗지피티가 추천한 크롤러 폴더 구조임 나중에 변동 가능

project-root/
│
├─ crawler/                      ← 🔍 크롤링 전용 폴더
│   ├─ agents/                   ← 플레이그라운드 페이지별 크롤링 로직 (ex: 학과별 등)
│   │   ├─ crawlMajorCourses.ts
│   │   └─ crawlAllMajors.ts
│   ├─ parsers/                  ← raw DOM → structured object로 변환
│   │   └─ parseCourseRow.ts
│   ├─ utils/                    ← 유틸 (스크롤 처리, 대기, 에러 핸들링 등)
│   │   └─ scrollVirtualTable.ts
│   ├─ types/                    ← 타입 정의
│   │   └─ course.ts
│   ├─ index.ts                  ← 크롤링 실행 진입점
│   └─ config.ts                 ← URL, 타임아웃 등 설정값
│
├─ db/                           ← DB 연결 및 저장 관련
│   ├─ client.ts
│   ├─ insertCourse.ts
│   └─ models.ts
│
├─ scripts/                      ← 전체 실행 스크립트
│   └─ crawlAndSave.ts          ← 크롤 + 저장 통합 실행
│
├─ .env                          ← DB 비번 등 환경 변수
├─ package.json
├─ tsconfig.json
